Oppsum:
Verben är få och läggs endast till verb från grupp I. Kan också andra kontinuativ/frekventativändelser kombineras med dessa verb?

De finns två omljud som uppträder. Ett a-omljud och ett e-omljud.
e - a => kolonne 2, R1 -> R2 e-e    -   bertedh -> bartehtjidh ->
ae - aa => kolonne 5 R1 -> R2 e-e   -   vaedtsedh -> vaadtsehtjidh ->

u - o => kolonne 3 R1 -> R4 e-e     -   buvvedh -> bovvehtjidh ->
ue - oe => kolonne 7 R1 -> e-e      -   vuetjedh -> voetjehtjidh ->

hur det blir med verb med stam på /i/ eller /ie/ finns inga exempel på. När det gäller /åe/ blir det /åa/ avsett vilket omljud som uppträder.
Att se dem i ljuset av liknande ändelser är det viktiga. Där kan vi hitta systemet.
